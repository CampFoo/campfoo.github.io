Add a custom sitemap to the sitemap index generated by Yoast plugin (WP SEO)

7th October 2015

Yoast's WP SEO plugin creates a handy sitemap index automatically, but what if you need to add more indices to this main index? There will be times when you'll have an extra bit of content that needs to be mapped, for example content that comes from another framework or source on the server but needs to be reflected in your root sitemap.

Fortunately there is a filter for this plugin that allows this, it's just a bit tricky to find. The filter is called "wpseo_sitemap_index". You may also need to prevent caching for a while in order to make the change appear. For this you need to filter "wpseo_enable_xml_sitemap_transient_caching" and return false.

// ../theme/functions.php
// Make sure we're using WPSEO
if( class_exists('WPSEO_Sitemaps') ) {
    function filter_wpseo_sitemap_index(){
        // Add custom item to sitemap index
        $sitemap_custom_items = '
            <sitemap>
               <loc>' . get_bloginfo('url').'/extra-sitemap.xml</loc>
               <lastmod>' . $last_date . '</lastmod>
            </sitemap<
        ';
        return $sitemap_custom_items;
    }
    add_filter( 'wpseo_sitemap_index', 'filter_wpseo_sitemap_index' );
    add_filter( 'wpseo_enable_xml_sitemap_transient_caching', '__return_false' );
}
